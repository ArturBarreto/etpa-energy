⚡ Energy API

A Spring Boot 3 / Java 21 application for managing load profiles, meters, cumulative readings, and deriving monthly consumption.

About

This API was built as part of the ETPA Energy use-case assignment. It demonstrates a clean architecture with:

  • Profiles with monthly fractions
  • Meters linked to profiles
  • Meter readings with yearly imports (12 values)
  • Consumption calculations per month
  • Dev mode: H2 in-memory DB, Prod mode: PostgreSQL on AWS RDS

Docs & Tools

  • README
  • Swagger UI – interactive docs
  • OpenAPI JSON – raw schema
  • H2 Console (dev only)

Database Info

Dev: H2 (in-memory)

  • JDBC: jdbc:h2:mem:testdb;MODE=PostgreSQL;DB_CLOSE_DELAY=-1
  • User: sa, password: (blank)
  • Console: /api/h2-console

Prod: PostgreSQL on AWS RDS (credentials from env vars)
